NVIDIA just turned every major creative tool into an MCP endpoint.

Blender, Unreal, Houdini, Affinity, Adobe — all MCP-ready at SIGGRAPH today. An agent can inspect scenes, rename layers, batch export, validate against pipeline rules. Same protocol Anthropic shipped for files and databases, now talking to your renderer.

This is the connector play, not the model play. The agent doesn't have to be smarter — it just has to speak the same protocol your tools do. That's the whole thesis of building a harness instead of swapping models.

Adobe, Epic, SideFX, and Boris all signed up. Not because they want agents doing creative work — because they want agents doing the 80% of production work that isn't creative. Renaming, validating, packaging, resizing, checking. The stuff nobody invoices for.

The protocol that wins is the one that connects the most tools. MCP just gained about 100 million daily seats.
