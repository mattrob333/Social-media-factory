Dario Amodei: software goes to ~zero marginal cost. Value moves to whoever wields it best.

True, but incomplete. The real bottleneck won't be wielding the tool — it'll be who can make the output reliable enough to bet on.

Free code has no warranty. Free agents have no audit trail. The value doesn't move to the prompter — it moves to whoever builds the harness: context, governance, and the human who signs for it when it breaks.

The model got cheap. Trust is still expensive.
