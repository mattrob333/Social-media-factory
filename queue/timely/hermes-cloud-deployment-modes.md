Hermes Cloud is live — hosted instances connecting via the desktop app or messenger gateways.

I've been running Hermes self-hosted since launch, and the question everyone asks is "why not cloud?"

Simple: when your agent touches enterprise permission boundaries, audit trails, or client data, self-hosted is the compliance floor. Least privilege means I know exactly where the agent runs, what it can reach, and who'd see the logs if a regulator called.

Cloud is the right call for personal use, team collaboration, or any deployment where the cost of a misconfigured permission is annoyance, not a breach.

The important thing: both options now exist as first-class paths in the same ecosystem. That's not a compromise — it's the minimum viable deployment architecture for serious agent work. Pick by workload, not by dogma.
