## Tue Jul 1 — LinkedIn Post

**Source:** Greg Isenberg — agent-native organizations
**Pillar:** Governance / NHI
**Remix mode:** amplify

```
Your next employee won't have a face.

Greg Isenberg wrote about "agent-native organizations" this week — companies structured so AI agents can run them. Agents with Slack logins, email accounts, history. This isn't a prediction. It's happening.

I fly a CRJ ~18 days a month and build AI agents the rest. Same job, honestly. Both fail quietly. Both punish skipping the checklist.

The conversation about agent-native companies focuses on speed and leverage. Nobody focuses on the blast radius.

Every agent you deploy is a non-human identity. Credentials. Permissions. System access. In most orgs, it will have no manager, no review, and no entry on the compliance dashboard.

We built identity governance for humans over 20 years. Then we handed system access to software that spins up faster than HR can blink and decided governance was someone else's problem.

If you're building for this future, you need three things before you need a better model: an owner for every agent, least-privilege by default, and an audit trail you'd hand to a regulator.

The agent-native org is coming. The breach that follows it won't have a face. Build the harness before you scale the fleet.

#AIGovernance
```

## Wed Jul 2 — X Thread

**Source:** Defensible theses — "harness over model"
**Pillar:** Prompt / context engineering
**Remix mode:** extend

```
The model is not your bottleneck.

I've watched frontier models faceplant because the harness fed them garbage. And I've watched "dumber" models win because someone engineered the input.

Stop shopping for models. Start building harnesses.
```

```
A harness is three things:

1. Context that actually represents the problem — not the prompt you wish would work, the real data with real edge cases.

2. Verification — a separate model that checks the output before it reaches a human or a system.

3. State — memory that survives the conversation. Without it, every interaction is ground zero.
```

```
The model is the engine. The harness is the steering, the brakes, the checklist.

Aviation figured this out 90 years ago. The plane matters. But what keeps you alive is the system around the plane.

AI teams are relearning this the hard way. You don't need GPT-5. You need to stop feeding your current model garbage.
```
