# Newsletter Drafts

Loop 3 (Weekly Newsletter) writes newsletter drafts here every Friday.

## File format: `YYYY-MM-DD.md`

Structure follows the voice.md newsletter spec:
1. TAKE OF THE WEEK — best POV expanded to 200-400 words
2. ON THE RADAR — 3-5 items with Matt's one-line take
3. ONE YOU CAN USE — prompt/framework readers can apply
4. BUILDING IN PUBLIC — shipping update (optional)

## Example

```markdown
# The Forward Deploy — June 27, 2026

## Take of the Week
[200-400 word essay]

## On the Radar
- **@karpathy on reasoning models** — The diminishing returns are real, but
  the enterprise take is different: most companies haven't even deployed the
  first generation yet. [link]
- ...

## One You Can Use
[Prompt or framework]

## Building in Public
[One line update]
```
