5 Prompt / Framework of the Week From the desk
The 20-Task Private Eval
1. Pull the last 20 real tasks your team gave an AI.
2. For each, write the input + one line of “what good looks like.”
3. Run all 20 through your current model and the candidate.
4. Blind-grade A/B — no model names visible.
5. Switch only on a clear win (≥70% preferred). Ties go to the incumbent.
Use before any model switch — or any time a launch thread makes a benchmark claim you’re tempted to believe.
