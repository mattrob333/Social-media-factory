# Post Drafts — Generated Content

Loop 2 writes generated post drafts here before scheduling. Each draft is
ready-to-post content in Matt's voice.

## File format: `YYYY-MM-DD.md`

Each post includes:
- Source item reference
- Platform (x / linkedin)
- Post type (thread / single / article)
- Full post text
- Attribution (if needed)
- Metricool schedule time (once queued)

## Example

```markdown
## X Thread — 2026-06-28 10:00 EST

**Remix mode:** amplify
**Attribution:** quote-tweet @GregIsenberg

they're missing the enterprise version of the same pattern.

level. You sit in their Slack, touch their real data, learn their real pain
points.
```
