Hermes Cloud is live — hosted instances connecting via the desktop app or messenger gateways.

I've been running Hermes self-hosted since launch, and the question everyone asks is "why not cloud?"

Simple: when an agent reads a client's Salesforce data behind a SAML gate, or touches a wire-transfer approval, self-hosted is the compliance floor. I know exactly which box the agent runs on, what network it can reach, and whose name is on the log if a regulator calls.

Cloud is the right call for personal use, team collaboration, or any deployment where the cost of a misconfigured permission is annoyance, not a breach.

Both options now exist as first-class paths in the same ecosystem. Pick by workload, not by dogma.
