# X (Twitter) Thread Template

The loop uses this as a format reference when generating X threads.

---

## Thread Format

A thread is 3-5 posts. Threads perform best when each post can stand alone
but the sequence tells a complete story.

### Structure

**Post 1 — Hook (the scroll-stopper)**
- One sharp sentence or question
- Makes someone stop scrolling
- Can be a bold claim, counterintuitive fact, or provocative question
- No context yet — make them click "Show more"

**Posts 2-4 — The meat (insights, steps, data)**
- Each post: one idea, fully expressed
- Use line breaks between sentences
- Numbers and specifics over generalities
- If it's a framework: one step/factor per post

**Post 5 (optional) — CTA or summary**
- One call to action: follow, reply, check the link
- Or a summary that ties the thread together
- Credit the source: "via @creator"
